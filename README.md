# Follow-Up OS

Never lose money from forgotten leads again.

A simple lead pipeline tracker for small business owners who sell through WhatsApp, Instagram, and email.

## What it does
- Add leads with deal value and follow-up date
- See who needs a message today
- Track your pipeline (Hot/Warm/Cold)
- Never forget a lead again

## Tech Stack
Python • FastAPI • SQLite • Railway

## Try it
[follow-up-os.up.railway.app](https://follow-up-os.up.railway.app)

## Who it's for
Freelancers, WhatsApp vendors, real estate agents, consultants — anyone who sells through conversations.
